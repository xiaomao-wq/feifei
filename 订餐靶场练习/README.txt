一、系统介绍
网上订餐系统
	功能模块：
			订餐：店铺和菜品的浏览、加入购物车
			订单：下单、支付
			会员：添加收货地址、信息修改、收藏店铺、评论
						
	流程：
		1.会员注册、登录
		2.去用户中心添加收货地址、修改会员头像等信息
		3.浏览店铺、收藏店铺、查看菜品
		4.在菜品页添加购物车
		5.购物车选中需要的食品，点击结算
		6.订单页选择收货地址、支付方式，提交订单
		7.选择支付或者进入用户中心点击“我的订单”，在订单列表对订单进行“取消订单”或者“支付”操作
		8.在订单列表点击订单编号可以进入订单详情页，查看食品、数量、价格等信息，未付款可以修改收货地址
		9.支付后，在订单列表点击“确认收货”，即表示订单已完成
		10.点击用户中心的“我的评论”，可以对已完成的订单进行评论
		11.评论完成后可以在对应的菜品、店铺页面中查看评论信息
		12.点击用户中心的“我的收藏”，查看或者删除收藏的店铺
